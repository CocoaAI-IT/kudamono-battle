# キャラクター・ビジュアル設計

Kudamono Battle は、擬人化した果物たちが台所のステージで戦う2Dプラットフォームファイターです。入力体系はスマホでも扱いやすいように、左スティック方向と `通常攻撃` / `必殺技` の組み合わせで構成します。

## 初期ロスター

| キャラクター | タイプ | 役割 |
| --- | --- | --- |
| Strawberry Samurai | スピード剣士 | 素早い差し込み、対空、復帰 |
| Banana Brawler | 中重量ブロウラー | 近距離の殴り合い、横撃墜 |
| Grape Gunner | 遠距離ゾーナー | 弾で相手を動かす |
| Watermelon Tank | 超重量級 | 遅いが一撃と耐久が強い |
| Pineapple Trapper | 設置/罠 | 床や足場を制圧する |
| Cherry Ninja | 高機動ラッシュ | 短リーチの連続攻撃 |

## Strawberry Samurai

スピードと剣の間合いで戦う基本キャラです。軽く撃墜力は控えめですが、機動力と復帰が強く、初心者でも扱いやすい立ち位置です。

- 武器: シアン色のベリーエネルギーブレード
- 強み: 移動速度、対空、復帰、差し込み
- 弱み: 軽さ、横方向の撃墜力
- ステータス: 移動 `420` / ジャンプ `655` / 空中制御 `0.60` / 重さ `0.96`

| 入力 | 技名 | 役割 |
| --- | --- | --- |
| ニュートラル通常 | Seed Jab | 素早い近距離小技 |
| 横通常 | Berry Blade | 剣で横を差す主力技 |
| 上通常 | Leaf Upper | 対空と着地狩り |
| 下通常 | Low Vine Cut | 足元への差し込み |
| ニュートラル必殺 | Pectin Pulse | 少し広い押し返し |
| 横必殺 | Jam Dash | 横移動を兼ねた突進 |
| 上必殺 | Leaf Lift | 復帰兼対空 |
| 下必殺 | Jam Spike | 下方向への強打 |

- 強み: 横撃墜、近距離火力、耐久

## 今後の候補

- Peach Grappler: 掴み・拘束・投げを中心にした近接圧力キャラ。
- Pear Glider: 空中制御と復帰力に寄せたフローティキャラ。

この2体は、掴みシステムや空中移動の追加設計が固まってから実装すると役割が被りにくくなります。

## アセット方針

キャラクター画像は `public/assets/characters` に配置し、各キャラが以下の6モーションを持ちます。

- `idle`
- `run`
- `jump`
- `quick`
- `heavy`
- `hurt`

キャラ選択画面用のポートレートは `public/assets/ui` に配置します。既存2体はWebP、追加4体は生成画像を透明背景PNGに加工した軽量アセットです。

攻撃エフェクトは `public/assets/effects` に配置します。内部の当たり判定は矩形のまま保持しますが、画面上には判定矩形を表示せず、キャラクター別の透明PNGエフェクトを表示します。

各キャラクターは次の6種のエフェクトを持ちます。

- `quick`: 通常攻撃向けの小さな斬撃・打撃
- `heavy`: 必殺技向けの大きな斬撃・打撃
- `projectile`: 飛び道具
- `trap`: 設置罠・床制圧
- `armor`: アーマー付き突進や防御的な技
- `combo`: 多段ヒットやラッシュ技
