# Smabura

A browser-based 2D platform fighter prototype built with Vite, TypeScript, and Phaser.

## Run

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:5173/`.

## Controls

- `A` / `D`: move
- `W`: jump / double jump
- `J`: quick attack
- `K`: heavy attack
- `L`: dodge
- `R`: restart

## Current Scope

- 1P vs CPU
- 3-stock match
- Damage-scaled knockback
- Fixed cyber dojo stage
- Original character and stage assets
